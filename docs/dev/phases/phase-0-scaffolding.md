# Fase 0: Scaffolding del Proyecto

## Objetivo
Crear la estructura Maven multi-módulo con Spring Boot BOM, Java 21/25 profiles, Jacoco, Surefire.

## Tareas
- Configurar estructura de directorios del proyecto Maven multi-módulo
- Crear POM raíz y POMs de módulos
- Configurar perfiles de Maven para Java 21 y 25
- Configurar integración con Jacoco para cobertura de código
- Configurar plugin de Surefire para ejecución de tests
- Crear archivos de configuración base (.gitignore, etc.)

## Componentes
- Estructura de directorios base
- Archivo POM raíz
- Archivos POM de módulos
- Configuración de compilación multi-JDK

## Criterios de Aceptación
- Estructura de proyecto Maven funcional
- Compilación exitosa en Java 21 y 25
- Integración con herramientas de análisis de código
